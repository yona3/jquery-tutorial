# jQuery 入門

簡単なTodoアプリ。

### 試すこと

- DOM操作
- イベント
- メソッド
- ajax
- ページ遷移
- TypeScriptの導入

### 使用技術

- jQuery
- TypeScript
- Webpack
- json server

### 環境構築

依存関係のインストール
```
$ yarn
```

### 開発
json server起動
```
$ yarn serve // localhost:3000
```

Webサーバー起動
```
$ yarn dev // localhost:8080
```

ビルド
```
$ yarn build
```
